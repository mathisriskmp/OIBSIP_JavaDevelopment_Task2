# OIBSIP_JavaDevelopment_Task2
## Project Title Online Examination System 
## Objective
To develop a Java-based Online Examination System that allows users to log in, attend an online exam, and view their results. 
## Steps Performed 
- User login authentication
  - Displayed multiple-choice questions
     - Accepted user answers - Calculated score
        - Displayed final result
  ## Tools Used
  - Java
  - VS Code
  - JDK 17
  ## Outcome
   The application successfully conducts an online examination, evaluates answers, and displays the final score.
